IOS project using Swift

his App is built following the instructions of Prof John Gallaugher. It is designed to be a Weather Info App that offers users weather information including: temperature, precipitation possbility and weather prediction in the following week. The API used in the App is DarkSky API.
Features:
  • Decoding JSON data from OpenWeather API
  • Google Place Autocomplete
  • Creating collection views, page view controllers, and converting time to different time zone
  • Converting the weather symbols with personal pictures
